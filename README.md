# Jogo Genius com HTML, CSS e JavaScript

## Disciplina: Programação Website I

## Tema: Frontend Inteligente com JavaScript

## Projeto: Criação do Jogo Genius

---

# 1. Objetivo do trabalho

Desenvolver uma versão web do jogo **Genius**, utilizando:

- HTML;
- CSS;
- JavaScript;
- manipulação do DOM;
- eventos de clique;
- arrays;
- funções;
- estruturas condicionais;
- lógica de repetição.

---

# 2. O que é o jogo Genius?

O Genius é um jogo de memória.

O sistema mostra uma sequência de cores.

O jogador deve repetir a sequência na mesma ordem.

A cada rodada:

1. o jogo adiciona uma nova cor;
2. mostra a sequência completa;
3. o jogador tenta repetir;
4. se acertar, passa de fase;
5. se errar, o jogo termina ou reinicia.

---

# 3. Etapas obrigatórias do trabalho

O trabalho deverá ser desenvolvido em **3 etapas**.

---

# ETAPA 1 — HTML

## Objetivo

Criar a estrutura visual do jogo.

O aluno deverá criar:

- título do jogo;
- área do tabuleiro;
- quatro botões coloridos;
- botão iniciar;
- área para mensagens;
- área para mostrar a rodada atual.

# ETAPA 2 — CSS

## Objetivo

Criar uma interface visual bonita, moderna e organizada.

O tabuleiro deve ter:

- quatro cores;
- efeito de clique;
- efeito de brilho;
- layout responsivo;
- aparência profissional.

# ETAPA 3 — Lógica básica com JavaScript

## Objetivo

Criar uma primeira versão funcional, onde:

- o botão iniciar começa o jogo;
- uma cor aleatória é escolhida;
- a cor pisca;
- o jogador clica em uma cor;
- o sistema informa se acertou ou errou.

# ETAPA 4 — Lógica completa

## Objetivo

Criar o jogo Genius completo, onde:

- o jogo cria uma sequência;
- a cada rodada adiciona uma nova cor;
- o sistema mostra a sequência completa;
- o jogador deve repetir corretamente;
- se acertar, avança de rodada;
- se errar, reinicia.
